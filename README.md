# 门店营业额预测工具

基于1165家门店数据的机器学习模型，用于门店选址决策支持。

## 🚀 快速访问
应用已部署到 Streamlit Cloud

## 📋 功能特点
- 智能选址评分系统
- 精准营业额预测
- 可视化分析图表
- 业务建议和投资回报分析

## 🛠️ 技术栈
- **前端**: Streamlit, Plotly
- **数据处理**: Pandas, NumPy
- **部署**: Streamlit Cloud

## 📁 项目结构
```
.
├── app.py              # 主应用文件
├── requirements.txt    # Python依赖
├── .streamlit/         # Streamlit配置
│   └── config.toml    # 主题配置
├── README.md          # 项目说明
├── simple_test.py     # 功能测试
└── test_app.py        # 完整测试
```

## 🔧 本地开发
```bash
pip install -r requirements.txt
streamlit run app.py
```
